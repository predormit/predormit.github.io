CSAPP Lab 1 Datalab

解题思路

第一题 bitXor

~(x&y) 筛选出了全为1的位，将其赋上0

~(~x&~y) 筛选出了全是0的位，将其赋上0

在&一下而得到异或的结果

第二题 tmin

把一移到符号位即可、

第三题 isTmax

TMAX + 1 = TMIN ,而TMAX与TMIN异或结果为0

再除去x是-1这种可能性即可

第四题 alloddbits

做法就是通过&和^来判断奇数位上是否全为1

但如果对x进行移位操作让a = 0xAA就超操作数了

所以干脆改a,通过位运算将其变成0xAAAAAAAA

第五题 negate

取反就是这样的

第六题 isAsciiDigit

本来想四位四位比较，后来觉得后四位有点难搞，没必要

后来有定义了三个变量来存x - (-0x30)这些数，但运行后提示变量 is unused,就只能写成一行了

第七题 conditional

肯定要先对x进行处理，将0仍旧变为0，但不等于0的数变成全一，然后就好做了

所以先逻辑运算，将其都变成0或1，由上题知~0 + 1 还是 0 ，而~x + x是全一

第八题 islessorequal

和ascii那题同理

第九题 logicalneg

因为对0取反还是0，0|0的符号位还是0，而其他数与相反数的或的符号位肯定为1，所以取反右移即可。

但因为是算数右移，所以0会得到负一，对其再一次取反就行

第十题 howmanybits

一个一个测可能会超操作数，所以用二分的思想，16，8，4，2，1，位的测试

最重要的一点就是先把他全部取正，所以需要取符号位对其判断

第十一题 floatscale2

浮点数有多种情况，所以要做的首先是取出阶码，符号位和尾数，再对阶码分情况讨论，不难看出，阶码全为0的时候左移即可，但同时要保留符号位，若全是1，则返回自身，else 阶码加一

第十二题 float2int

同理，先取出阶码，符号位和尾数，再分情况讨论。

同样处理符号位

取尾数的时候前面加个1
